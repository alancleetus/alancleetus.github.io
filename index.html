SOME OF MY WORK

Big Num Calculator

Source at GitHub


This application converts equations in infix notation to postfix notation. The resulting postfix notation is used to calculate the final result. The postfix method is used to make all calculations. This program can also do calculations on numbers of datatype bigNum (a big numbers is same as bigInteger from Java). This was also my final project for CSC 20: Programming Concepts and Methodology II.

schoolPad

Source at GitHub


Notepad application I created in Java to learn more about I/O functionalities in Java. This application allows users to read/write/edit txt files. Also comes with a toggleable school mode which converts all text to font "Times New Roman", and font size "12".
