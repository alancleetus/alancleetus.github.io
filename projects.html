Attendance Tracking Web Application

Developed a database and related web pages used to track the amount and types of students attending events hosted by the Association of Computing Machinery chapter at CSUS (ACM). This application works with the CSU Sacramento student ID card, enabling ACM officers to sign attendees into events using a magnetic card reader. Application was built using HTML5, CSS3, Bootstrap, JavaScript, PHP, MySQL, jQuery, and AJAX.
